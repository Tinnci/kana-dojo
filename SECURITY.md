# Security Policy

Kana Dojo is an early-stage learning app and does not currently process accounts, payments, or remote user data.

## Supported Versions

Only the latest commit on `main` is maintained.

## Reporting A Vulnerability

Please open a private security advisory on GitHub when available, or open an issue with minimal reproduction details if the report does not expose sensitive information.

Include:

- affected version or commit
- Android version and device if relevant
- steps to reproduce
- expected and actual behavior
- any logs that do not include private data
